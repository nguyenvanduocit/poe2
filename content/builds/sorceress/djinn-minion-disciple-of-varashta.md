# Djinn Minion Disciple of Varashta

## Quick Start

Build summoner dùng **3 Djinn bất tử** + **Raging Spirits** với **Purple Fire** (chaos damage nhắm vào resistance thấp nhất, 2x damage lên ES). League starter hoàn hảo, không cần unique, fully online ở Act 3.

### Cheatsheet

```
┌─────────────────────────────────────────────────────────────────┐
│ ACT 1:  Unearth → Fireball → Living Bomb → Raging Spirits      │
│         Rotation: LB → LB → FB → repeat                         │
├─────────────────────────────────────────────────────────────────┤
│ ACT 2:  ★ Blackflame Covenant (Purple Fire keystone)           │
│         ★ Trial 1: 2 Djinn (Kelari + Ruzhan)                   │
```

## Core Mechanics

> **Hiểu 5 hệ thống này = chơi build hiệu quả.**

### 1. Purple Fire (Blackflame Covenant)

```
Fire Spell → Chaos Damage (màu tím)
           → 2x damage lên Energy Shield
           → Hit resistance THẤP NHẤT (không phải Fire res)
```

**Ví dụ:** Enemy có Fire 75%, Chaos -10% → Normal fire hit 75% res (25% damage), Purple Fire hit -10% res (110% damage).

### 2. Muster Support - Damage Multiplier

```
Mỗi loại permanent minion KHÁC NHAU = +7% MORE damage (multiplicative)

3 Djinn                    = 21% MORE
+ 1 Sniper + 1 Warrior + 1 Mage = 42% MORE

- 10 Snipers = 7% MORE (1 loại)
- 1 mỗi loại = 42% MORE (6 loại)
```

**Stack nhiều Muster gems:** 1 gem = 1.42x | 2 gems = 2.02x | 3 gems = 2.86x

### 3. Raging Spirits - Main Damage

```
Cast Fire Spell → Auto spawn Raging Spirits (max 2/cast)
               → Max 10-11 spirits, duration 5 giây
               → Không thể điều khiển

Ngừng cast = Spirits hết hạn = Damage về 0
```

### 4. Casting Rotation

```
Living Bomb → Living Bomb → Fireball → [repeat]
     ↓              ↓            ↓
+2 Spirits    +2 Spirits    +2 Spirits + 12 firebolts
+remnant      +remnant      (tiêu thụ remnants)
```

**Ngừng level LB & FB ở Lv13** - Spells chỉ là trigger, damage từ minions.

### 5. Kelari's Deception - Boss Burst

```
Use 1: 100% damage + tạo clone
Use 2: 150% damage + clone 1 nổ
Use 3: 200% damage + clone 2 nổ
─────────────────────────────────
Tổng: 450% damage trong ~2.4 giây
```

**Giữ 3 charges cho boss**, dùng rapid-fire.

## Progression Timeline

### Act 1

| Milestone | Action |
|-----------|--------|
| Start | Lấy Unearth |
| Level 3 Uncut | Cut Fireball (main damage) |
| Level 3 Uncut #2 | Cut Living Bomb |
| Spirit Gem | Cut Raging Spirits |
| **Goal** | Thiết lập rotation LB→LB→FB |

**Gear:** Wand với +Fire Spell Levels hoặc %Spell Damage

### Act 2

| Milestone | Action |
|-----------|--------|
| **Blackflame Covenant** | Lấy NGAY khi có (keystone) |
| Elemental Weakness | Curse cho bosses |
| **Trial 1** | Barya of Kelari + Barya of Ruzhan |
| **Goal** | Purple Fire active + 2 Djinn |

**Gear:** Chuyển sang Scepter với +Minion Skills

### Act 3 (BUILD ONLINE)

| Milestone | Action |
|-----------|--------|
| **Trial 2** | Barya of Navira (Djinn #3) |
| **Kelari's Deception** | Boss killer unlocked |
| Muster Support | Farm nhiều, socket vào mọi minions |
| **Goal** | 3 Djinn = 21% MORE, burst damage ready |

**Gear:** Tìm Rattling Scepter (free Skeletal Warriors)

### Acts 4-10

| Milestone | Action |
|-----------|--------|
| **Lv13 cap** | Ngừng level LB & FB |
| Interlude 3 | +40 Spirit từ Kriar Village |
| Skeletal diversity | 1 Sniper + 1 Warrior + 1 Mage |
| **Goal** | 6 unique minions = 42% MORE |

**Gear priorities:**
1. +Minion Skill Levels (Scepter, Helm, Amulet)
2. +Spirit (cho minion diversity)
3. Mana Leech (QUAN TRỌNG - không có = không spam được)
4. Life, Resistances 75%

**Target Act 10:** 2000+ life, 75% res, 150+ Spirit, 6 minion types
