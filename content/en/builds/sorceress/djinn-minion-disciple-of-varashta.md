# Djinn Minion Disciple of Varashta

## Quick Start

A summoner build using **3 immortal Djinn** + **Raging Spirits** with **Purple Fire** (chaos damage that targets the lowest resistance, 2x damage against ES). A perfect league starter, no uniques required, fully online by Act 3.

### Cheatsheet

```
┌─────────────────────────────────────────────────────────────────┐
│ ACT 1:  Unearth → Fireball → Living Bomb → Raging Spirits      │
│         Rotation: LB → LB → FB → repeat                         │
├─────────────────────────────────────────────────────────────────┤
│ ACT 2:  ★ Blackflame Covenant (Purple Fire keystone)           │
│         ★ Trial 1: 2 Djinn (Kelari + Ruzhan)                   │
├─────────────────────────────────────────────────────────────────┤
│ ACT 3:  ★ Trial 2: Djinn #3 (Navira) + Kelari's Deception      │
│         ★ Farm Muster Support gems                              │
│         BUILD FULLY ONLINE                                      │
├─────────────────────────────────────────────────────────────────┤
│ 4-10:   Stop leveling LB & FB at Lv13 | Stack Spirit           │
│         Interlude 3: +40 Spirit | Target 6 minion types         │
├─────────────────────────────────────────────────────────────────┤
│ CLEAR:  Living Bomb → Living Bomb → Fireball → repeat          │
│ BOSS:   Curse → Kelari ×3 (rapid) → Rotation                   │
└─────────────────────────────────────────────────────────────────┘
```

### Pros / Cons

| Pros | Cons |
|----|-------|
| Zero-cost league starter | Must cast constantly (rhythm-based) |
| High clear + Boss damage | Have to manage Spirit |
| Purple Fire bypasses all resistance | Glass cannon if you're careless |
| Djinn never die | Not a passive summoner |

## Core Mechanics

> **Understand these 5 systems = play the build effectively.**

### 1. Purple Fire (Blackflame Covenant)

```
Fire Spell → Chaos Damage (purple color)
           → 2x damage against Energy Shield
           → Hits the LOWEST resistance (not Fire res)
```

**Example:** an enemy with Fire 75%, Chaos -10% → a normal fire hit lands against 75% res (25% damage), Purple Fire hits against -10% res (110% damage).

### 2. Muster Support - Damage Multiplier

```
Each DIFFERENT type of permanent minion = +7% MORE damage (multiplicative)

3 Djinn                    = 21% MORE
+ 1 Sniper + 1 Warrior + 1 Mage = 42% MORE

- 10 Snipers = 7% MORE (1 type)
- 1 of each type = 42% MORE (6 types)
```

**Stacking multiple Muster gems:** 1 gem = 1.42x | 2 gems = 2.02x | 3 gems = 2.86x

### 3. Raging Spirits - Main Damage

```
Cast Fire Spell → Auto spawn Raging Spirits (max 2/cast)
               → Max 10-11 spirits, duration 5 seconds
               → Cannot be controlled

Stop casting = Spirits expire = Damage drops to 0
```

### 4. Casting Rotation

```
Living Bomb → Living Bomb → Fireball → [repeat]
     ↓              ↓            ↓
+2 Spirits    +2 Spirits    +2 Spirits + 12 firebolts
+remnant      +remnant      (consumes remnants)
```

**Stop leveling LB & FB at Lv13** - the spells are just triggers, the damage comes from the minions.

### 5. Kelari's Deception - Boss Burst

```
Use 1: 100% damage + creates a clone
Use 2: 150% damage + clone 1 explodes
Use 3: 200% damage + clone 2 explodes
─────────────────────────────────
Total: 450% damage over ~2.4 seconds
```

**Hold 3 charges for the boss**, then rapid-fire them.

### System Interaction

```
        MUSTER SUPPORT (×1.42+)
               │
    ┌──────────┼──────────┐
    ▼          ▼          ▼
 3 DJINN    RAGING     SKELETAL
(permanent) SPIRITS    ARMY
   │       (temp)     (permanent)
   │          │          │
   └────┬─────┴─────┬────┘
        ▼           ▼
   LIVING BOMB   FIREBALL
        │           │
        └─────┬─────┘
              ▼
      PURPLE FIRE (Chaos)
```

## Progression Timeline

### Act 1

| Milestone | Action |
|-----------|--------|
| Start | Grab Unearth |
| Level 3 Uncut | Cut Fireball (main damage) |
| Level 3 Uncut #2 | Cut Living Bomb |
| Spirit Gem | Cut Raging Spirits |
| **Goal** | Set up the LB→LB→FB rotation |

**Gear:** Wand with +Fire Spell Levels or %Spell Damage

### Act 2

| Milestone | Action |
|-----------|--------|
| **Blackflame Covenant** | Take it RIGHT AWAY when available (keystone) |
| Elemental Weakness | Curse for bosses |
| **Trial 1** | Barya of Kelari + Barya of Ruzhan |
| **Goal** | Purple Fire active + 2 Djinn |

**Gear:** Switch to a Scepter with +Minion Skills

### Act 3 (BUILD ONLINE)

| Milestone | Action |
|-----------|--------|
| **Trial 2** | Barya of Navira (Djinn #3) |
| **Kelari's Deception** | Boss killer unlocked |
| Muster Support | Farm a lot, socket into every minion |
| **Goal** | 3 Djinn = 21% MORE, burst damage ready |

**Gear:** Find a Rattling Scepter (free Skeletal Warriors)

### Acts 4-10

| Milestone | Action |
|-----------|--------|
| **Lv13 cap** | Stop leveling LB & FB |
| Interlude 3 | +40 Spirit from Kriar Village |
| Skeletal diversity | 1 Sniper + 1 Warrior + 1 Mage |
| **Goal** | 6 unique minions = 42% MORE |

**Gear priorities:**
1. +Minion Skill Levels (Scepter, Helm, Amulet)
2. +Spirit (for minion diversity)
3. Mana Leech (IMPORTANT - without it you can't spam)
4. Life, Resistances 75%

**Target Act 10:** 2000+ life, 75% res, 150+ Spirit, 6 minion types

## Endgame Transition

### Chaos Inoculation Build

**When to switch:** After campaign, once you have 5000+ ES from gear

```
CI Benefits:
├─ Immune to Chaos Damage
├─ Immune to Bleeding
└─ The Fourth Teaching synergy (-1s ES recharge delay)
```

### Ascendancy Order

| Trial | Nodes | Priority |
|-------|-------|----------|
| 1 | Barya of Kelari + Ruzhan | Mandatory |
| 2 | Barya of Navira + Kelari's Deception | Mandatory |
| 3 | The Fourth Teaching | High (CI synergy) |
| 4 | Sacred Rituals or Baryanic Leylines | Situational |

### Passive Tree Focus

**Campaign:** Minion Damage → Spirit → Gigantic Following → Blackflame Covenant

**Endgame:** Chaos Inoculation → ES nodes → ES recharge → Minion survivability

### Target Stats

| Phase | Life/ES | Spirit | Minion Types |
|-------|---------|--------|--------------|
| Act 10 | 2000+ Life | 150+ | 6 |
| Early Maps | 3000+ ES | 200+ | 6 |
| Endgame | 6000+ ES | 350+ | 6 |

**Q: Is this HC viable?**

A: None. Purple Fire bypasses resistance issues.
