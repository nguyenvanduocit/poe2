# Djinn Commander - Dook's Version

## Quick Start

The build centers on the **two main Djinn** (Kelari + Ruzhan) using their Command abilities. The Djinn cost NO Spirit → spend the leftover Spirit on Skeletal minions via Muster Support. It's a high-APM playstyle with burst damage coming from Kelari's Deception.

### Cheatsheet

```
┌─────────────────────────────────────────────────────────────────┐
│ DJINN SETUP:                                                     │
│   Kelari (Sand): Brutality (AoE) + Deception (Burst ×3)         │
│   Ruzhan (Fire): Fury (immobilized bonus) + Passive Exposure    │
├─────────────────────────────────────────────────────────────────┤
│ CORE MECHANICS:                                                  │
│   ★ Djinn = 0 Spirit cost (free!)                               │
│   ★ Excess Spirit → Skeletal minions via Muster                 │
│   ★ "At your Command" converts Spirit → damage                  │
│   ★ Gigantic Following = +20% more minion damage                │
├─────────────────────────────────────────────────────────────────┤
│ ASCENDANCY ORDER:                                                │
│   1. Barya of Kelari → 2. Kelari's Deception →                  │
│   3. Barya of Ruzhan → 4. Ruzhan's Fury                         │
├─────────────────────────────────────────────────────────────────┤
│ ROTATION:                                                        │
│   Clear: Kelari's Brutality spam (low CD)                       │
│   Boss:  Brutality ×2 → Deception ×3 → Ruzhan's Fury            │
└─────────────────────────────────────────────────────────────────┘
```

### Pros / Cons

| Pros | Cons |
|----|-------|
| Djinn = 0 Spirit (free minions) | High APM requirement |
| Djinn are immortal, no need to invest in their survivability | Limited AoE coverage |
| High burst damage (Deception ×3) | Needs deliberate targeting |
| Low mana costs | Campaign/Leveling focus only |
| Flexible Spirit allocation | Needs a setup swap for Endgame |

## Core Mechanics

> **Djinn are free on Spirit → stack other minions through Muster**

### 1. Djinn Economy

```
Kelari + Ruzhan = 0 Spirit cost
        ↓
Excess Spirit → Skeletal Minions
        ↓
Muster Support: +7% MORE damage per unique minion
        ↓
Djinn damage scales with the number of minion types
```

**Key insight:** the Djinn cost no Spirit but still get buffed by Muster Support from your other minions.

### 2. Kelari Commands

```
Kelari's Brutality:
├─ Multi-hit AoE
├─ Low cooldown
└─ Primary clearing tool

Kelari's Deception:
├─ 3-stack physical burst
├─ +50% damage per stack
├─ Use 1: 100% → Use 2: 150% → Use 3: 200%
└─ Boss killer
```

### 3. Ruzhan Integration

```
Ruzhan's Fury:
├─ Fire volley
├─ Bonus damage on immobilized enemies
└─ Use when the boss is near its Stun Threshold

Ruzhan Passive:
├─ Triggers when you cast Unearth
└─ Applies Exposure (reduces ele res)
```

### 4. "At your Command" Passive

```
Excess Spirit → Persistent Minion Damage
        ↓
Each Skeletal minion = +7% MORE for the Djinn
        ↓
Goal: Collect 1 of EACH minion type
```

### 5. Gigantic Following

```
+20% MORE minion damage
BUT -1 minion slot

Trade-off: Worth it if you have enough minion diversity
```

### Act 1: Hybrid Start

| Milestone | Action |
|-----------|--------|
| Start | Bone Cage (primary) |
| Add | Unearth (raise minions) |
| Weapon | Wand + Sceptre (hybrid scaling) |
| **Goal** | High APM, build the minion army |

**Note:** "Accept slower progress if preferring pure minion experience"

### Act 2: Djinn Unlock

| Milestone | Action |
|-----------|--------|
| Trial of Sekhemas (Lv21-22) | Unlock Barya of Kelari |
| Point 2 | Kelari's Deception |
| Weapon swap | Wand → Shield (25% block) |
| **Goal** | Kelari = primary damage |

**Rotation after ascendancy:**
- Brutality for clearing
- Deception for single target

### Act 3: Full Djinn Setup

| Milestone | Action |
|-----------|--------|
| Trial of Chaos | 2 points (Ruzhan + Fury) |
| Minion collection | 1 of each Skeletal type |
| Muster Support | Socket into minions |
| **Goal** | Maximize "At your Command" |

### Act 4 & Interlude

| Milestone | Action |
|-----------|--------|
| Interlude | +40 permanent Spirit |
| Gear focus | Spirit, ES, +Minion Levels |
| **Goal** | Endgame transition prep |
