# Raging Spectre Shaman - Cocaine Wasps

## Quick Start

A summoner build using **2 types of Spectre** (Gargantuan Wasp for clear, Death Knight Elite for boss) with the **Rage sharing** mechanic from the Shaman ascendancy. The Spectres are extremely aggressive, fly through obstacles, and deal enormous damage. Nickname: "Cocaine Wasps" — the evolved version of the old "Cocaine Ants".

### Cheatsheet

```
┌─────────────────────────────────────────────────────────────────┐
│ SPECTRES:                                                        │
│   Clear:  Gargantuan Wasp (Ashen Forest - Interlude)            │
│   Boss:   Death Knight Elite (Holten - Interlude) [RARE only!]  │
├─────────────────────────────────────────────────────────────────┤
│ CORE MECHANICS:                                                  │
│   ★ Oil Commanding Rage onto the Amulet                         │
│   ★ Do NOT use Charms (Sacred Flow = +120 Spirit)               │
│   ★ Infernal Cry every 4 seconds to maintain Rage               │
│   ★ Weapon swap: Wasp (clear) ↔ Death Knight (boss)             │
├─────────────────────────────────────────────────────────────────┤
│ ASCENDANCY ORDER:                                                │
│   1. Sacred Flow → 2. Druidic Champion →                        │
│   3. Furious Wellspring → 4. Reactive Growth                    │
├─────────────────────────────────────────────────────────────────┤
```

## Core Mechanics

> **The build is based on Rage sharing — understand it well = high damage.**

### 1. Rage Sharing (Commanding Rage)

```
You gain Rage → Minions receive Rage (via Commanding Rage passive)
             → Minions gain Attack Speed from Rage
             → Damage output increases enormously
```

**Setup:**
- Oil **Commanding Rage** onto the Amulet (MANDATORY)
- The Shaman ascendancy node shares rage to minions

### 2. Sacred Flow - No Charms = +120 Spirit

```
Do NOT equip any Charm
        ↓
Sacred Flow ascendancy = +120 Spirit
        ↓
More Spirit = more minions/supports
```

**IMPORTANT:** Empty charm slots are a core mechanic, not a choice.

### 3. Spectre Strategy - Weapon Swap

```
Weapon Set 1: Gargantuan Wasp
├─ Insane clear speed
├─ Flies through obstacles
├─ Aggressive, finds enemies on its own
└─ Used for mapping

Weapon Set 2: Death Knight Elite
├─ Best single target in the game
├─ Must bind the RARE version (elite skill)
└─ Used for bosses

Weapon Lock: Right-click the I/II icons in inventory
```

### 4. Rage Generation - Infernal Cry

```
Infernal Cry + Enraged Warcry II + Raging Cry + Astral Projection
        ↓
Ranged nova warcry = Gain rage from a distance
        ↓
Use every 4 SECONDS to prevent rage decay
```

**Warning:** Rage decay = minions lose damage. Cast Infernal Cry frequently!

### 5. Minion Immortality - Tecrod's Revenge

```
Minions die (from Infernal Legion burn)
        ↓
Tecrod's Revenge triggers
        ↓
Temporary immortality after death
        ↓
Respawn and continue fighting
```

### System Interaction

```
      COMMANDING RAGE (Amulet Oil)
               │
               ▼
    INFERNAL CRY (Rage Gen)
               │
               ▼

GARGANTUAN WASP    DEATH KNIGHT ELITE
(Clear - Set 1)    (Boss - Set 2)
    │                     │
    └──────────┬──────────┘
               ▼
        WEAPON SWAP
    (Lock both sets active)
```

| Milestone | Action |
|-----------|--------|

| **Trial 4** | Reactive Growth (elemental defense) |

| **Goal** | Full rage sharing active |

**Gear priorities:**

2. Belt: Darkness Enthroned + Talisman of Ralakesh
3. Jewel: From Nothing (Bulwark) → Warlord Berserker

5. Focus: Energy Shield, Spirit

### FAQ

**Q: Why do I have to bind the RARE Death Knight?**
A: Only the RARE version has the elite skill (an extra ability). The normal Death Knight doesn't, and deals much less damage.

**Q: Why not use Charms?**
A: The Sacred Flow ascendancy gives +120 Spirit when no charms are equipped. Spirit > charm bonuses for this build.

**Q: Does spamming Infernal Cry cost mana?**
A: Warcries use a Rage cost, not Mana. Cast every 4 seconds to maintain rage on the minions.

**Q: Is it league start viable?**
A: Difficult. It needs specific items (Commanding Rage oil, From Nothing jewel). Better as a second build after you have currency.

**Q: How does weapon swap work?**
A: Lock both weapon sets (right-click I/II). Swap before a boss fight. Spectres are bound to the weapon set and aren't lost when you swap.
