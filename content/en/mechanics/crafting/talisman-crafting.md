# Talisman Crafting Guide - Budget to Min-Max

## Overview

The Talisman is the main weapon type for many physical-based attack builds in POE2, especially the [Meteor Bear Shaman](/builds/druid/meteor-bear-shaman/). This guide covers 5 crafting methods from 1 divine to 8+ divine investment, letting you craft a Talisman that fits your current budget and upgrade it gradually as you progress.

### Method Comparison

| Method | Cost | PDPS Range | Difficulty |
|--------|------|------------|------------|
| **Budget (Hybrid)** | ~15-25x | 450-550 | Easy |
| **Recombinator** | ~50-75x + gold | 550-650 | Medium |
| **Transmute Spam** | Variable | 500-600 | Medium |
| **Buy Magic Base** | ~3 divine | 550-650 | Easy |
| **Fracture Min-Max** | 5+ divine | 700+ | Hard |

### Base Selection

**IMPORTANT:** Always use the **Magi Talisman** (NOT the Fury Talisman) - the base damage is significantly higher.

**Item Level:** You need ilvl 75+ to roll T1 flat physical damage.

### Build Context

This guide is designed for physical-based attack builds, especially the [Meteor Bear Shaman](/builds/druid/meteor-bear-shaman/) that uses a Talisman as its main weapon.

Target Stats for Meteor Bear:
- High pDPS (550+)
- +Attack Skill Levels (Battle Essence)
- Attack Speed (Amamanu's Desecrate unveil)
- Gain Physical as Extra (endgame prefix)

## How It Works

### Method 1: Budget Hybrid (15-25x)

The cheapest method, suited for early mapping.

**Step-by-Step:**

1. **Buy a base:** Search for a Magic Magi Talisman with Hybrid %Physical. Filter: NOT accuracy (avoid hybrid accuracy). Cost: ~10x per base.

2. **Apply Abrasion:** Greater Essence of Abrasion for T3 flat physical. Cost: ~3x.

3. **Prefix Jawbone:** Add the Amamanu's Desecrate modifier. Unveil -> rolling for %Physical Damage.

4. **Repeat:** If you don't hit %phys -> buy a new base, repeat.

**Expected Results:**
- Hybrid %phys: ~80%
- Abrasion flat: T3
- Unveiled %phys: 60-100%
- Total: ~450-550 PDPS
- Cost per attempt: ~13-15x
- Average attempts: 3-5 for a decent %phys roll

**Finishing the Budget Base:**

Since the suffixes are full (from the Jawbone unveil), you only need:

1. **Perfect Battle Essence:** ~50c-1d, for +5 to Attack Skills
2. **Optional - Vaal Infuser:** 5/6 chance to add quality (no corrupt), push 20% -> 25%+ quality

### Method 2: Recombinator (~50-75x + gold)

The most cost-efficient method for high PDPS bases.

**Core Concept:** The Recombinator prioritizes keeping the base with the rarer mod. T1 flat phys (rare) + T4 %phys (common) -> likely keeps the base with flat phys.

**Setup:**

| Component | Requirements | Cost |
|-----------|-------------|------|
| **Base A (Magi Talisman)** | T1 or T2 flat physical damage, ilvl 75+ | 10-15x per base |
| **Base B (Any Talisman)** | T4 %physical damage (134%+), ilvl 75+, NOT accuracy | 1-3x per base |
| **Lucky Recombinator Omen** | Doubles success chance (~10% -> ~20%) | ~25x |

**Process:**

1. Put the T1 flat phys base (Magi) in the left slot
2. Put the T4 %phys base in the right slot
3. Use the Lucky Omen
4. Recombinate

**Success Rate:** ~20-25% with the Lucky Omen. Expected result: 550-650+ PDPS base.

**Common Mistakes:**
- Wrong item level: Base B needs to be able to roll T1 flat phys (ilvl 75+), otherwise -> reduced success chance
- Wrong base type: Buying a Fury Talisman instead of a Magi - Fury has lower base damage
- Hybrid accuracy mods: Always filter "NOT accuracy" - hybrid accuracy is not pure %phys

### Method 3: Transmute Spam (Variable)

Mass rolling for people who like gambling.

**Process:**

1. Buy 20-30 white Magi Talisman bases (~4x per base)
2. Perfect Transmute all bases
3. Perfect Augment all bases
4. Sort results:
   - T1 flat phys -> Save for the Recombinator
   - Hybrid %phys -> Budget method
   - Main %phys (RARE) -> Jackpot
   - Trash -> Scour or sell

**Expected Distribution (per 30 bases):**
- T1 flat phys: 2-3 bases (Recom fodder)
- Hybrid %phys: 4-6 bases (Budget craft)
- Main %phys: 0-1 bases (Best outcome)
- Trash: 20+ bases (Resale value)

### Method 4: Buy Magic Base (~3 divine)

Skip crafting, buy directly.

**Trade Search Filters:**
- Base: Magi Talisman
- Rarity: Magic
- %Physical Damage: 90%+ (T2+)
- Item Level: 75+
- Cost: ~3 divine for T1-T2 %phys

**Finishing:**

1. Apply Abrasion -> T3 flat phys
2. Suffix Exalt -> Force suffix
3. Dextral Essence Omen + Gain Phys Essence -> Force prefix
4. Amamanu's Desecrate -> Attack speed unveil
5. Battle Essence -> +5 attack skills
6. Vaal Infuser -> Quality 25%+

### Method 5: Fracture Min-Max (5+ divine)

Highest PDPS potential, most expensive.

**Base:** 3-socket Magi Talisman (ilvl 75+), ~20x per base.

**Target:** T2 %physical damage (Tyrannical) + T1 flat physical (via chaos spam after fracture).

**Process:**

1. Null + Aug until T2 %phys
2. Apply Abrasion -> T3 flat
3. Roll suffix with +5 attack speed craft
4. Add Amamanu's Desecrate (suffix)
5. Fracture (1/3 chance to hit %phys)
6. If success: Anul off suffixes except one, chaos spam for T1 flat phys
7. Finish with gain mod, battle essence, etc.

**Expected Result (Perfect craft):**
- T2 %phys (fractured): 160%+
- T1 flat phys: 61-90
- Gain physical as extra
- +5 Attack Skills
- Attack Speed 18%
- Total: 750-800+ PDPS

## Strategy & Tips

### Finishing Techniques

**Forcing Gain Physical as Extra:**

1. **Suffix Exalt:** Force a suffix slot (~5x)
2. **Dextral Essence Omen:** "Essence removes random suffix" (~165x)
3. **Perfect Gain Physical Essence:** Guarantees the gain phys prefix (~3x)
4. Apply the Dextral Omen, then apply the Gain Phys Essence - the Essence removes a suffix, adds gain phys

**Amamanu's Desecrate Unveil:**

Amamanu only has 3 unveil options:
- **Attack Speed (18%)** <- TARGET
- Spirit Reservation
- Fishers

Process: Desecrate with the Amamanu modifier -> Prefix Jawbone -> Unveil -> pick Attack Speed (1/3 guaranteed outcomes).

**Vaal Infuser Quality:**
- 5/6 chance: Add quality (no corrupt)
- 1/6 chance: Corrupts (stops at current quality)
- Strategy: Spam until corrupt, typical result 24-27% quality
- Each 1% quality = significant PDPS increase

### Panic Recovery

If you slam a bad suffix (like +melee levels):

- **Problem:** Battle Essence needs no +levels
- **Solution 1:** Amamanu's Desecrate (suffix) -> 1/3 to remove the bad mod
- **Solution 2:** Perfect Essence of Haste -> 50% to remove/replace
- **Warning:** May remove good mods, risky!

**Best Practices:**
- Always use the Magi Talisman, NOT Fury
- Filter "NOT accuracy" when buying bases
- Check ilvl 75+ before crafting
- Use the Lucky Recombinator Omen for the Recombinator method
- Battle Essence is the final finishing step
- Vaal Infuser for a quality boost with minimal risk

**Common Mistakes:**
- Buying a Fury Talisman instead of a Magi (lower base damage)
- Not filtering hybrid accuracy mods
- Wrong item level (need 75+ for T1 flat phys)
- Applying Battle Essence while a bad suffix is still on (wastes the essence)
- Rushing the Fracture method before you have enough budget
